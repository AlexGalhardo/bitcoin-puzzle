## Development Setup Local

Prerequisites:
   - Install Bun: https://bun.sh/docs/installation

1. Clone repository
```bash
git clone git@github.com:AlexGalhardo/xandyh-bitcoin-puzzle.git
```

2. Enter repository
```bash
cd xandyh-bitcoin-puzzle/
```

3. Install dependencies
```bash
bun install
```

4. Start script
```bash
bun start
```
